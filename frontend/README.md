# Frontend
This folder contains the frontend (user interface) for the Smart Parking Capacity Enforcement System.
## Purpose
The frontend provides a dashboard and user interface for both municipal authorities and citizens to view real-time parking data.
## Features
- Live parking capacity display
- Parking availability status
- Alerts for full parking zones
- Admin dashboard for monitoring violations
- Citizen interface for checking available parking
## Technologies Used
- React.js
- HTML
- CSS
- JavaScript
## Future Implementation
- Interactive parking map
- Real-time updates using API
- Mobile-friendly responsive design
